## Basic Usage
The `Dialog` component is designed to come with some default UI elements for the basic use cases. Passing certain properties will automatically render the default UI elements instead of having to pass your own.

### Default Title
The `Dialog` component will automatically render a default title if you pass a `title` prop. If you don't pass a `title` prop, the default title will not be rendered.

### Default Actions
The `Dialog` component will automatically render a default actions if you pass a `onSubmit` prop. If you don't pass an `onSubmit` prop, the default actions will not be rendered.

**Note:** That when passing default actions you also need to pass a translation function to the `Dialog` component that supports the following keys:

- `cancel`
- `submit`

You can pass a translation function to the `Dialog` component to override the default translations.

### Example: Simple Confirmation Dialog
```jsx
import {
  Dialog,
} from '@encodium/rp-ui';
import { useState } from 'react';

import useTranslation from 'src/hooks/use-translation';

const PageWithModal = () => {
  const [open, setOpen] = useState(false);
  const handleClose = () => {
    setOpen(false);
  };
  const handleSubmit = () => {
    setOpen(false);
  };

  // Supports the 'cancel' and 'submit' keys
  const { translate } = useTranslation({
    basePath: 'your-feature-name',
    namespace: 'your-feature-namespace',
  });

  return (
    <div>
      <button onClick={() => setOpen(true)}>{translate('openModal')}</button>
      <Modal
        open={open}
        onClose={handleClose}
        onSubmit={handleSubmit}
        translate={translate}
        message={translate('confirmationMessage')}
        title={translate('title')}
        size="small"
      />
    </div>
  );
};
```

## Advanced Usage
Even though we have the helper properties, a user can still take full control of the modal by defining their own title, content, and actions.

### Example: Custom Header
```jsx

import {
  Button,
  Dialog,
  DialogTitle,
  DialogContent,
  DialogActions,
  IconButton,
} from '@encodium/rp-ui';
import { Close } from '@mui/icons-material';

const MyDialog = ({
  open,
  onClose,
  onSubmit,
}) => {
  return (
    <Dialog
      open={open}
      size="small"
      onClose={onClose}
    >
      <DialogTitle>
        Dialog Title
        <IconButton onClick={onClose} size="small">
          <Close size="small" />
        </IconButton>
      </DialogTitle>
      <DialogContent>
        Dialog Content
      </DialogContent>
      <DialogActions>
        <Button onClick={onSubmit} color="primary" variant="contained">Action</Button>
      </DialogActions>
    </Dialog>
  );
};
```
